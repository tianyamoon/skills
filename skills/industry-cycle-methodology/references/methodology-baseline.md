## 一、总公式

`结构力量 -> 宏观金融条件 -> 产业供需变量 -> 公司回报质量 -> 市场定价 -> 市场心理 -> 阶段映射 -> 结论`

这条链路的意思不是每次都要写得很长，而是每次都要知道自己当前的判断停留在链路的哪一段。

## 三、五类周期

1. 行业周期
   - 看行业供需、渗透率、产能扩张、价格与库存。
   - 回答的是“这个行业现在处在什么阶段”。
2. 盈利周期
   - 看收入增速、毛利率、费用率、经营杠杆、现金流。
   - 回答的是“公司利润现在处在加速、见顶还是出清阶段”。
3. 价格周期
   - 看现货价、合约价、ASP、折扣、竞品报价、价差。
   - 回答的是“价格在反映短缺、抢装、去库还是价格战”。
4. 资本开支周期
   - 看扩产公告、在建工程、设备订单、融资能力、进入者数量。
   - 回答的是“今天的扩张会不会成为明天的过剩”。
5. 资金参与周期
   - 看估值、持仓拥挤、成交结构、主题热度、融资环境。
   - 回答的是“市场已经把多少预期提前交易掉了”。

默认不要把五类周期混成一句“景气度好”。它们常常同向，但并不总是同步。

## 四、默认分析顺序

1. 先定边界：
   行业到底指什么产品、什么环节、什么区域、什么客户。
2. 再定主变量：
   先找真正驱动利润和估值的 3-5 个核心变量。
3. 再看需求：
   谁买、为什么买、买多少、买多久。
4. 再看供给：
   谁供给、扩多快、进入门槛在哪里、替代路线是什么。
5. 再看价格和库存：
   价格是否验证供需，库存是在积累还是去化。
6. 再看资本开支：
   扩产是否可融资、可落地、可复制。
7. 再看公司财务：
   增长是否变成利润、利润是否变成现金流、回报率是否可持续。
8. 再看估值与预期：
   市场已经把什么写进价格里。
9. 最后才做阶段判断与行业映射。

## 五、九步流程

1. 判断问题类型
   - 是研究阶段
   - 研究公司组
   - 研究估值是否透支
   - 研究历史映射
2. 建变量表
3. 判主次变量
4. 进入传导链
5. 判行业阶段
6. 判资本反噬
7. 判预期透支
8. 判穿越公司
9. 做行业映射

## 六、强制回答的问题

1. 最底层驱动是什么？
2. 这个驱动属于结构力量、周期力量、政策力量还是金融条件变化？
3. 它如何传导到出货、价格、利润和估值？
4. 现在改善的是需求、供给、价格、库存、资本成本中的哪一项？
5. CapEx 是扩大战果还是埋下过剩？
6. 市场已定价多少预期？
7. 哪些证据已验证，哪些仍待补证？
8. 这个结论是行业结论还是公司结论？
9. 如何迁移到目标行业？

## 七、最常见错误

1. 先有观点再找资料。
2. 用股价上涨替代基本面验证。
3. 用单个明星公司替代整个行业。
4. 用高利润率外推长期高回报。
5. 忽略估值、拥挤和信用条件。
6. 忽略资本开支反噬。
7. 忽略失败样本和低端产能。
8. 不区分事实、判断和假设。
