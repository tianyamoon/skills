## 一、目的

这份文件解决的不是“怎么研究”，而是“研究结果怎样排布，才像顶级基金的正式输出”。

它默认承接的前提是：

1. 过程层已经基本完成。
2. 关键变量与证据链已经成型。
3. 主要判断已经有证伪条件和风险边界。

要点只有一句：

**顶级基金写作首先优化读者路径，而不是作者路径。**

作者的自然顺序通常是：

`搜资料 -> 做变量表 -> 看股价 -> 看财务 -> 想结论 -> 补风险`

但顶级基金给读者的顺序通常是：

`先告诉你结论和框架 -> 再给你成绩单/记分牌 -> 再解释发生了什么 -> 再展开证据 -> 再说明动作、风险和边界`

## 二、本次拆解的一手材料

以下材料均来自官方公开来源，观察重点不是观点本身，而是内容组织方式。

1. Berkshire Hathaway 2025 shareholder letter
   - 官方入口：[Berkshire Hathaway Shareholder Letters](https://www.berkshirehathaway.com/letters/letters.html)
   - 样本特征：用“文化、资本纪律、风险管理、运营原则”组织首封 CEO 年信。
2. GMO 2025 年第四季度 quarterly letter《It’s Probably a Bubble, But There Is Plenty Else to Invest In》
   - 官方入口：[GMO Research Library](https://www.gmo.com/americas/research-library/)
   - 样本特征：先给 executive summary，再给论证框架、历史对照、图表和结论。
3. Oaktree 官方备忘录《AI Hurtles Ahead》
   - 官方入口：[Howard Marks Memos / Oaktree Insights](https://www.oaktreecapital.com/insights)
   - 样本特征：教育型备忘录结构，先交代写作动机，再做概念澄清，再讲最新变化和含义。
4. Pershing Square Holdings 2025 Annual Report
   - 官方入口：[Pershing Square Holdings Reports](https://www.pershingsquareholdings.com/company-reports/)
   - 样本特征：先 company overview 和 performance，再 chairman statement、investment manager report、principal risks、governance、financials。

## 三、核心结论

### 1. 顶级基金先给“前门”，再给“仓库”

他们不会一上来就把原始研究全倒给读者，而是先搭前门。

高频“前门”包括：

1. 一句话核心判断
2. 业绩或阶段记分牌
3. 本期最重要的变化
4. 本文会回答什么问题

### 2. 他们把不同功能的内容分开，而不是混写

常见功能层：

1. `成绩单`
   - 先说结果、相对表现、阶段位置、关键指标。
2. `解释层`
   - 解释为什么会这样，哪些变量在起作用。
3. `动作层`
   - 说今年做了什么、加了什么、减了什么、为何如此。
4. `风险层`
   - 说什么可能错、什么仍未确认。
5. `附录层`
   - 放表格、注释、会计口径、补充图表。

### 3. 他们先给“可压缩结论”，再给“不可压缩细节”

读者应该先看 1 页就明白 70%，再用后面的 10 页确认剩下 30%。

这意味着：

1. 结论必须能独立成立。
2. 每个大节都要有自己的功能，而不是堆信息。
3. 图表、历史对照、公司明细要服务于主判断，而不是抢主判断。

### 4. 他们明确区分“框架性文字”和“研究性文字”

1. Berkshire/Pershing 的主席信、CEO 信常承担：
   - 价值观
   - 资本配置原则
   - 风险边界
   - 年度动作说明
2. GMO/Oaktree 的季度信/备忘录常承担：
   - 当期主题判断
   - 历史类比
   - 关键变量与图表
   - 投资含义

这两种写法不能混成一个口径。

## 四、四种高频组织原型

### A. 主席信 / 年度总述型

代表样本：

1. Berkshire 2025 shareholder letter
2. Pershing Square Holdings 2025 Chairman’s Statement

适用场景：

1. 模块年度收口
2. 总论总述
3. 结项交接材料
4. 需要先交代框架、原则、动作、边界的长文

建议结构：

1. 开场摘要
2. 经营/研究框架
3. 本期关键动作
4. 关键结论
5. 风险与未解决问题
6. 下一阶段优先事项

### B. 主题季度信 / 市场观点型

代表样本：

1. GMO 4Q 2025 quarterly letter
2. Oaktree《AI Hurtles Ahead》

适用场景：

1. 某个主题的集中表达
2. 阶段判断更新
3. 某一轮行情或泡沫的解释
4. “为什么我们现在这样看”的高可读说明文

建议结构：

1. Executive Summary
2. 写作缘起 / 本文主问题
3. 论证框架
4. 历史类比或概念澄清
5. 核心证据与图表
6. 结论与投资含义
7. Appendix

### C. 年报分层型

代表样本：

1. Pershing Square Holdings 2025 Annual Report

适用场景：

1. 大模块对外版整理
2. 需要兼容“快速阅读”和“深度复核”的正式成果
3. 需要同时服务管理层、研究者、后续接手者

建议结构：

1. Overview
2. Performance / Scorecard
3. Chair Summary / Executive Summary
4. Main Research Body
5. Principal Risks and Uncertainties
6. Governance / Method / Definitions
7. Financials / Data Appendix / Sources

### D. 原则与方法信型

代表样本：

1. Berkshire 2025 letter 的 culture / integrity / capital discipline / risk management 分节

适用场景：

1. 方法论总纲
2. 研究框架升级
3. 团队协作规则
4. 需要先统一“我们怎么看”和“我们怎么做”的文档

建议结构：

1. 这份框架解决什么问题
2. 我们坚持什么原则
3. 这些原则如何落到研究动作
4. 反例与边界
5. 如何用于后续项目

## 五、迁移到本技能的落地规则

1. 先给读者一个“前门摘要”。
2. 强制加入“记分牌”。
3. 把“事实、判断、动作、风险”拆开。
4. 把“变化”单独拎出来。
5. 让附录承担“仓库功能”。
6. 默认假设读者没参与本轮研究，只给你 3 到 5 分钟。

## 六、常见失误

1. 把研究过程顺序当成读者阅读顺序。
2. 一上来堆原始资料，没有前门摘要。
3. 把事实、判断、风险、动作混写，读者抓不到主线。
4. 没有记分牌，导致读者无法快速定位阶段。
5. 没有“本期变化”，导致材料难以连续跟踪。
6. 结论写很长，但缺一条可压缩的一句话判断。
7. 把所有数据都放正文，导致主判断被淹没。
8. 只写支持论据，不写相反证据和证伪条件。
