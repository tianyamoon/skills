# wiki修链

## 目标

修复知识页面之间的断链问题，让 wiki 从孤点页面恢复成可导航、可追踪的关系网络。

## 优先修复的关系

1. 主题 -> 主题
2. 主题 -> 实体
3. 实体 -> 主题
4. 复盘 / 总结 -> 主题或实体
5. 方法 / SOP -> 相关页面

## 修复规则

1. 先补高价值链接，不追求一次全连满。
2. 优先补强关系，不是见到同名词就互链。
3. 能双向链接的尽量双向。
4. 链接要服务检索、导航和复用，而不只是让图谱好看。
5. 如果页面还不稳定，先标成待补链候选，不硬写死。
