# 页面模板

## 题材页模板

建议至少有：

1. `## 定义`
2. `## 当前催化`
3. `## 产业链/分支`
4. `## 代表股`
5. `## 关联题材`
6. `## 相关复盘`
7. `## 证据来源`

## 个股页模板

建议至少有：

1. `## 公司定位`
2. `## 题材归属`
3. `## 关键催化`
4. `## 风险点`
5. `## 关联页面`
6. `## 相关复盘`
7. `## 证据来源`

## 复盘页模板

建议至少有：

1. `## 市场判断`
2. `## 主线与分支`
3. `## 重点个股`
4. `## 风险提示`
5. `## 次日计划`
6. `## 关联节点`
7. `## 证据回挂`

## 方法页模板

建议至少有：

1. `## 适用场景`
2. `## 核心定义`
3. `## 判断标准`
4. `## 执行动作`
5. `## 常见误区`
6. `## 相关页面`

## 模板使用原则

1. 按需补齐，不要机械灌满。
2. 先保证页面能回答问题，再追求完整度。
3. 缺少证据的内容标 `待核`。
