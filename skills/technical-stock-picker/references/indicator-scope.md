# 当前版本指标范围

## 一、版本边界

这个技能第一版只覆盖我们当前已经在用的技术指标。

默认范围内：

1. `月线趋势`
2. `周线趋势`
3. `6月线乖离率`
4. `12月线乖离率`
5. `5日 / 20日 / 60日` 强弱
6. `YTD`
7. `距60日高点`
8. `量价结构`
9. `支撑位 / 压力位`
10. `执行位 / 证伪位`

默认不覆盖：

1. `MACD`
2. `RSI`
3. `布林带`
4. `缠论`
5. `波浪`
6. `盘口逐笔`

用户若要这些扩展，再单独升级技能。

## 二、月线趋势

默认看三个条件，满足其中两项及以上，可写 `月线向上`：

1. 最新月收盘价 `>= 6月均线`
2. `6月均线 >= 上月6月均线`
3. 最新月收盘价 `>= 12月均线`

若只满足一项，写 `月线待确认`。  
若三项里零项满足，写 `月线不合格`。

这个定义是资格线，不是买点。

## 三、周线趋势

默认看三个条件，满足其中两项及以上，可写 `周线向上`：

1. 最新周收盘价 `>= 20周均线`
2. `20周均线 >= 上周20周均线`
3. 最近 `4` 周里，至少 `3` 周收盘在 `20周均线` 上方

若只满足一项，写 `周线待确认`。  
若三项里零项满足，写 `周线不合格`。

## 四、月线乖离率

公式：

1. `6月线乖离率 = 当前价 / 6月均线 - 1`
2. `12月线乖离率 = 当前价 / 12月均线 - 1`

若当月月 K 未收盘，用最新收盘价对动态月均线估算，并标注 `动态月线口径`。

当前纪律阈值：

1. `6月线乖离率 > 60%`：原则上不新增主动买入
2. `12月线乖离率 > 100%`：原则上不新增主动买入
3. `6月线乖离率 40%-60%`：只能等分歧承接或平台消化后再评估

## 五、日线执行层

### 1. 强弱字段

默认计算：

1. `5日涨跌幅`
2. `20日涨跌幅`
3. `60日涨跌幅`
4. `YTD涨跌幅`
5. `距60日高点`

公式：

1. `N日涨跌幅 = 当前收盘 / N个交易日前收盘 - 1`
2. `距60日高点 = 当前收盘 / 近60日最高价 - 1`

### 2. 解释纪律

1. 月线、周线都没过关时，日线强弱只作观察，不作主动买入理由。
2. `20日` 和 `60日` 更适合判断中短期结构。
3. `距60日高点` 更适合识别高位加速、分化、滞涨和回撤。

## 六、量价结构

当前版本只做简化判断，不做复杂形态学扩展。

默认看：

1. `5日平均成交量 / 20日平均成交量`
2. `5日平均成交额 / 20日平均成交额`
3. 上涨时是否放量、下跌时是否缩量

简化解释：

1. `量比 > 1` 且价格走强：可写成 `放量强化`
2. `量比 < 1` 但价格仍稳：可写成 `缩量整理`
3. 放量下跌：优先写成 `承接转弱`

## 七、支撑、压力、证伪

当前版本先用最稳定的简化口径：

1. `20日低点` 作为近端支撑
2. `60日高点` 作为近端压力
3. `20日低点失守` 作为日线第一证伪位

如果用户明确要周线或月线支撑位，再额外说明口径。

## 八、输出解释纪律

1. `月线向上 + 周线向上 + 乖离不过大`：才有资格进入主动买入候选
2. `月线向上 + 周线向上 + 乖离过大`：写成 `持有审计 / 等回踩 / 等平台消化`
3. `月线向上 + 周线未确认`：写成 `等待右侧`
4. `月线未向上`：只能写成 `观察 / 修复 / 回避`

技术面只是资格和执行层，不是产业逻辑本身。
