# 外部方法论校准

这个文件用来解释：为什么这套选股标准是这样设计的。

## 核心结论

第二版标准不是凭感觉拼出来的，而是把四种资金语言分开使用：

1. `机构语言` 解决估值、现金流、ROIC、资本周期
2. `私募语言` 解决风格漂移、弹性和组合风险
3. `量化语言` 解决趋势、拥挤、容量和模型失效
4. `游资方法论` 解决短周期买点、卖点、仓位和退潮纪律

四种语言都重要，但谁也不能替代谁。

## 四种语言分别贡献什么

| 资金语言 | 主要解决什么 | 不能替代什么 |
|---|---|---|
| `机构主导` | 长期利润池、现金流、估值预期、资本周期 | 短线执行点 |
| `私募主导` | 产业趋势、公司弹性、风格风险、组合容错 | 公告和财报核验 |
| `量化主导` | 趋势、因子、拥挤、容量、模型衰减 | 产业逻辑 |
| `游资主导` | 情绪周期、主线地位、辨识度、标准买卖点 | 经营验证 |

## 强采纳条款

下面这些应当视作强规则：

1. 先排方向，再排个股
2. 经营验证大于故事
3. 买可验证路径，不买概念标签
4. 必须反推估值预期
5. 高景气方向必须审计资本周期反噬
6. 月线和周线只作为门槛，不作为逻辑本体
7. 月线乖离率需要单独审计
8. 必须说明当前是谁在主导

## 条件采纳条款

下面这些更适合放在执行层：

1. 标准买点
2. 标准卖点
3. 退潮降仓或空仓纪律
4. 量化拥挤和容量警报

## 不采纳为正式买因的内容

以下内容不能升级成正式买入理由：

1. `龙虎榜席位标签`
2. 自然人身份传闻
3. 某游资“买了所以我也买”
4. 单日热点热度

## 对用户解释时的简版口径

可以这样解释：

1. 机构语言回答“这个方向值不值得研究和配置”。
2. 私募语言回答“这个方向当前弹性和风格是否合适”。
3. 量化语言回答“这个趋势是不是太拥挤、太脆弱”。
4. 游资方法论回答“短周期执行是不是标准、是不是该撤”。

这种分层解释，通常比泛泛说“这个方向好”更扎实。
