# 第二版选股标准

这个文件负责给出核心决策链和硬性纪律。

## 固定决策链

统一按下面顺序判断：

`方向排序 -> 可验证路径 -> 经营验证 -> 估值预期 -> 资金属性 -> 技术门槛 -> 排除法 -> 反方条件 -> 执行条件`

不要反过来。

## 保留的九条纪律

必须完整保留：

1. `方向优先于个股`
2. `经营验证大于故事`
3. `高位强票，不无脑追，审慎评价`
4. `低位方向不能因为没涨就买`
5. `必须有反方条件`
6. `买的是可验证的路径，不是概念标签`
7. `月线向上`
8. `周线向上`
9. `月线乖离率不过大`

## 每一环必须回答什么

### 方向排序

先判断方向是：

1. `主攻`
2. `备选`
3. `观察`
4. `回避`

必须解释：

1. 为什么排在这个位置。
2. 排序依据来自经营变量、组级强弱、资金强弱，还是只有题材热度。
3. 如果是低位方向，为什么不能仅仅因为没涨就上调。

### 可验证路径

把概念拆成传导链，比如：

1. `订单 -> 收入 -> 利润 -> OCF`
2. `价格上行 -> 毛利率改善 -> 利润兑现`
3. `CapEx 增加 -> 设备 / 基建订单 -> 交付 -> 回款`
4. `客户导入 -> 份额提升 -> 规模效应 -> ROIC 改善`

说不清路径的，只能留在观察池。

### 经营验证

优先使用以下证据：

1. 订单或合同
2. 收入和利润
3. OCF、回款、合同负债、应收、存货
4. 客户导入或价格变量
5. 公告、财报、交易所回复、可靠行业数据

不要把政策口号、同业映射、情绪热度当成经营验证。

### 估值预期

必须反问：

1. 当前价格已经隐含了几年高增长。
2. 当前估值隐含的利润率和 ROIC 是否可持续。
3. 远期订单和远期利润是否已经被提前打满。
4. 下一次财报如果不及预期，还剩多少容错。

### 资金属性

明确当前是谁在主导：

1. `机构主导`
2. `私募主导`
3. `量化主导`
4. `游资主导`

资金属性用于解释上涨方式和退潮方式，不是买入理由本身。

### 技术门槛

主动买入候选必须满足：

1. `月线向上`
2. `周线向上`
3. `月线乖离率不过大`

初始阈值：

1. `6月线乖离率 > 60%`：原则上不新增主动买入
2. `12月线乖离率 > 100%`：原则上不新增主动买入
3. `6月线乖离率 40%-60%`：等待分歧承接或平台消化后再评估
4. 日线只负责择时，不得替代月线和周线

### 排除法

只要命中硬排除项，即使方向很强，也要降级。

### 反方条件

必须写清楚什么情况说明判断错了，而且要能被观察到。

### 执行条件

最后才回答：

1. 什么情况下允许进攻
2. 什么情况下只能观察
3. 还缺什么右侧确认
4. 什么情况下必须放弃、降级或退出

## 硬排除清单

统一使用下面这些排除条件：

1. 没有经营验证
2. 估值透支
3. 月线乖离率过大
4. 低位无趋势
5. 高位无消化
6. 短线热度没有基本面
7. 游资方法论买点不成立
8. 游资方法论卖点已触发
9. 量化或短线资金退潮
10. 资金属性错配
11. 资本周期反噬
12. 重资产方向遇到信用收缩
13. 只有概念标签，没有可验证路径
14. 没有反方条件

## 实务提醒

这套标准不是说：

1. 最强股都能追
2. 低位股都值得买
3. 叙事能代替公告和财报
4. 日线强就能覆盖月周线

它说的是：只有 `路径 + 经营验证 + 估值预期 + 资金属性 + 技术资格` 同时成立，才有资格进入主动买入候选。
